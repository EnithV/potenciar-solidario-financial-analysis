### **Advanced DAX Measures Library**

#### **Core Financial Metrics**
```dax
Total_Revenue = 
SUM('Donors'[Importe])

Total_Expenses = 
SUM('Vendors'[Importe])

Net_Utility = 
[Total_Revenue] - [Total_Expenses]

Utility_Margin = 
DIVIDE([Net_Utility], [Total_Revenue], 0)

YoY_Revenue_Growth = 
VAR CurrentYear = [Total_Revenue]
VAR PreviousYear = 
    CALCULATE(
        [Total_Revenue],
        SAMEPERIODLASTYEAR('Calendar'[Date])
    )
RETURN
    DIVIDE(CurrentYear - PreviousYear, PreviousYear, 0)
```

#### **Advanced Analytics Measures**
```dax
Donor_Retention_Rate = 
VAR CurrentDonors = 
    DISTINCTCOUNT('Donors'[Número])
VAR PreviousYearDonors = 
    CALCULATE(
        DISTINCTCOUNT('Donors'[Número]),
        SAMEPERIODLASTYEAR('Calendar'[Date])
    )
VAR RetainedDonors = 
    CALCULATE(
        DISTINCTCOUNT('Donors'[Número]),
        FILTER(
            'Donors',
            'Donors'[Número] IN VALUES('PreviousYear'[Donors])
        )
    )
RETURN
    DIVIDE(RetainedDonors, PreviousYearDonors, 0)

COVID_Impact_Adjustment = 
VAR COVID_Period = 
    DATESBETWEEN('Calendar'[Date], DATE(2020,3,1), DATE(2022,12,31))
VAR Normal_Revenue = [Total_Revenue]
VAR COVID_Revenue = 
    CALCULATE([Total_Revenue], COVID_Period)
RETURN
    Normal_Revenue - COVID_Revenue

Inflation_Adjusted_Expenses = 
VAR Base_Expenses = [Total_Expenses]
VAR Inflation_Rate = 
    IF(YEAR(MAX('Calendar'[Date])) = 2024, 1.15, 1.0)
RETURN
    Base_Expenses * Inflation_Rate

Vendor_Concentration_Risk = 
VAR Top5_Vendors = 
    SUMX(
        TOPN(5, VALUES('Vendors'[Número Proveedor]), [Total_Expenses]),
        [Total_Expenses]
    )
RETURN
    DIVIDE(Top5_Vendors, [Total_Expenses], 0)
```

#### **Predictive Analytics**
```dax
Revenue_Forecast_Next_Quarter = 
VAR Historical_Average = 
    AVERAGEX(
        DATESINPERIOD(
            'Calendar'[Date],
            MAX('Calendar'[Date]),
            -12,
            MONTH
        ),
        [Total_Revenue]
    )
VAR Seasonal_Factor = 
    DIVIDE(
        [Total_Revenue],
        CALCULATE(
            [Total_Revenue],
            SAMEPERIODLASTYEAR('Calendar'[Date])
        ),
        1
    )
RETURN
    Historical_Average * Seasonal_Factor

Sustainability_Score = 
VAR Utility_Stability = 
    1 - STDEV.P([Monthly_Utility]) / AVERAGE([Monthly_Utility])
VAR Donor_Diversity = 
    1 - [Donor_Concentration_Risk]
VAR Financial_Health = 
    IF([Utility_Margin] > 0.5, 1, [Utility_Margin] * 2)
RETURN
    (Utility_Stability + Donor_Diversity + Financial_Health) / 3
```

## 🚀 Getting Started

### **Prerequisites**
- Microsoft Power BI Desktop (latest version)
- Excel 2016+ (for source data review)
- 8GB RAM recommended (4GB minimum)
- Access to Potenciar Solidario context (provided in documentation)

### **Quick Start Guide**

1. **Repository Setup**:
   ```bash
   git clone https://github.com/EnithV/potenciar-solidario-financial-analysis.git
   cd potenciar-solidario-financial-analysis
   ```

2. **Data Verification**:
   - Review `data/historical_analysis/potenciar_solidario_report.pdf`
   - Validate insights consistency with Looker Studio baseline
   - Examine `data/BD_Sprint_3_PowerBI.xlsx` for source data

3. **Power BI Dashboard**:
   - Open `powerbi/Potenciar_Solidario_Dashboard.pbix`
   - Allow data refresh when prompted
   - Navigate using the page tabs (Executive → Revenue → Expenses → Analysis)

4. **Interactive Exploration**:
   - Start with Executive Summary for validated KPIs
   - Use year/country filters to explore specific segments
   - Click visuals for cross-filtering and drill-down analysis
   - Export reports using Power BI's native functionality

5. **Customization Guide**:
   - Replace sample data with your organization's financial records
   - Modify DAX measures in `scripts/dax_measures_library.txt`
   - Adjust visual themes using `assets/brand_guidelines/`
   - Configure automated refresh using `scripts/powerbi_automation.py`
